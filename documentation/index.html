Description

Slak is a simple, creative and responsive HTML template for showcase any creative and multipurpose information. We used all modern technologies like HTML5, CSS3, bootstrap and jQuery in this template. The template is 100% responsive and supports all new and old browsers.All the codes are clean and well organized.

Fonts and Typography

Slak uses Google Web Fonts. To change titles, just change the link located within "index.html" marked "https://fonts.googleapis.com/css?family=lato|poppins'" and open "css/style.css" to change the headings located in the body styles.

CONTACT FORM

Slak includes a simple form mailer script to provide you with an instant contact form. It ships with the theme package as "assets/php/form-process.php" and the only requirement is a PHP5-capable web server. We recommend PHP 5.1.0 or higher.

You can set up the recipient email address through the configuration interface. Simply enter a valid email address in the "assets/php/form-process.php" files and change it into "$EmailTo = "yourmail@mail.com";"
